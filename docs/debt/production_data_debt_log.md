# Production data debt log

DD-series. Forward-looking enterprise data risk, distinct from the data-quality issues in `data_debt_log.md` (the data is not wrong, it is not yet sourced for production). The Lean MVP synthesises its snapshot; production deployments face a different data shape. Agentic capability is de-risked; data sourcing, completeness, and per-firm customisation are the real long-term enterprise risk. Restructured out of the combined `PRODUCT_DEBT_LOG.md`; see `README.md` for the convention.
