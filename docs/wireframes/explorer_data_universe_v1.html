Boundaries this page holds (audit Part A7) The case screen is not touched: links into a case are plain navigation, no Case Detail components or styles change (WA29 single-writer). Forward snapshots always carry the DM3 disclosure. Every investor row carries its tier badge; rows with tier real_local_only (none exist today) suppress identity fields by rule. Snapshot refresh, exports, firm configuration editing, and any tag or curation surface are out of scope (audit Part B).
